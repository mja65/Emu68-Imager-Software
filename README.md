This is the repository for the Emu68 Imager tool for version 2 and up. The most recent version of the tool is found under the "Releases" [tab](https://github.com/mja65/Emu68-Imager-Software/releases). Simply download the latest version, extract, and run the Emu68Imager.cmd file.

Please note, that commits in the repository which are not included in a release should be considered a Work in Progress! They may cause issues or just not work at all. You have been warned!

Full documentation is available [here](https://mja65.github.io/Emu68-Imager/).
